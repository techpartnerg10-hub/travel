# Travel Next.js Project

temp 프로젝트를 기반으로 한 Next.js 여행 사이트 프로젝트입니다.

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 프로젝트 구조

```
project/
├── app/              # Next.js App Router 페이지
├── components/       # React 컴포넌트
├── public/          # 정적 파일 (이미지, 폰트 등)
├── styles/          # CSS 스타일 파일
└── lib/             # 유틸리티 함수
```

## 참고

- temp 프로젝트의 HTML 구조와 스타일을 Next.js 컴포넌트로 변환
- temp 프로젝트의 CSS/JS 파일들은 public 또는 styles 폴더에서 참조
